# PhishGuard Trainer - Phishing Attack Simulator

An interactive phishing simulation and training platform designed to educate users about email-based cyber threats. This project helps organizations improve their security posture by testing and training employees to recognize and respond to phishing attacks.

## Overview

PhishGuard Trainer simulates realistic phishing scenarios in a safe, controlled environment. Users learn to identify suspicious emails, malicious links, and social engineering tactics through hands-on practice. The platform provides immediate feedback and detailed explanations to reinforce learning.

## Features

- **Realistic Phishing Simulations**: Authentic-looking email scenarios mimicking real-world attacks
- **Interactive Training Modules**: Hands-on learning experience with immediate feedback
- **Multiple Attack Types**: Coverage of various phishing techniques and social engineering tactics
- **Progress Tracking**: Monitor learning advancement and identify areas for improvement
- **Detailed Explanations**: In-depth breakdowns of red flags and security indicators
- **Safe Learning Environment**: Practice without risk to real systems or data
- **Performance Analytics**: Insights into user responses and common mistakes

## Technologies Used

- **Frontend**: Next.js, React, TypeScript, Tailwind CSS
- **Backend**: Node.js, Server Actions
- **Database**: PostgreSQL (for tracking user progress and results)
- **Deployment**: Vercel
- **UI Components**: Custom interactive email interface

## Phishing Scenarios Covered

The simulator includes various phishing attack types:

- Credential harvesting attempts
- Malicious attachment scenarios
- CEO fraud / Business Email Compromise (BEC)
- Spear phishing campaigns
- Urgent action social engineering
- Brand impersonation attacks
- Link manipulation technique

- ## Project Goals

- Reduce successful phishing attacks through education
- Build a security-conscious organizational culture
- Provide measurable security awareness training
- Identify vulnerable users requiring additional training
- Demonstrate ROI of security awareness programs

## Impact & Results

- **Improved Detection Rates**: Users learn to identify phishing attempts more accurately
- **Reduced Click-Through Rates**: Fewer employees fall victim to simulated attacks over time
- **Behavioral Change**: Measurable improvement in security-conscious decision making
- **Risk Reduction**: Lower organizational vulnerability to real phishing attacks
